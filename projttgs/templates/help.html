How to Use Timetable Generator

Follow these 7 simple steps to generate your timetable

1

Add Department

Start by adding your department details. This includes the department name and other details.

2

Add Teachers

Add all teachers in your department. Include their names, ID according to the institution's rules and regulations.

3

Add Courses

Input all courses that need to be scheduled. Include course codes, names, weekly slots, course type etc accordingly.

4

Add Timings

Define the available time slots for classes. Set up your daily schedule according to slots per day.

5

Add Rooms

Add all available classrooms and their capacities. Include type of room for specific rooms.

6

Add Sections

Create sections for each course. Where you need to assign teachers, subjects, years etc according to the institution.

7

Generate Timetable

Click the generate button to create your optimized timetable. The system will automatically assign teachers, rooms, and time slots while considering all constraints.
